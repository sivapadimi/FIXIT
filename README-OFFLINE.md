# FixIt Debugging Platform - Offline Mode 🐛

## 🚀 Quick Start for Event Participants

### Method 1: Easy Launch (Recommended)
1. **Double-click**: `START-FIXIT.bat`
2. **Wait**: Server starts automatically
3. **Open browser**: Goes to platform automatically

### Method 2: Manual Launch
1. **Open Command Prompt**
2. **Navigate**: `cd "path-to-FixIt-folder"`
3. **Run**: `python FixIt-Offline-Server.py`

## 🌐 How to Access

### For Event Host:
- **Local URL**: http://localhost:8080
- **Network URL**: http://[YOUR-IP]:8080

### For Other Participants:
1. **Connect** to same WiFi/network
2. **Open browser**
3. **Go to**: http://[HOST-IP]:8080
4. **Start debugging!**

## 🎯 Troubleshooting

### Server Won't Start:
- **Check Python**: Make sure Python is installed
- **Port Conflict**: Change PORT in script if needed
- **Permissions**: Run as administrator if required

### Can't Access from Other Devices:
- **Firewall**: Allow Python through firewall
- **Network**: Ensure same WiFi/network
- **IP Address**: Use correct host IP

### Code Editor Issues:
- **Browser**: Use modern browser (Chrome, Firefox, Safari)
- **JavaScript**: Enable JavaScript
- **Cache**: Clear browser cache if needed
